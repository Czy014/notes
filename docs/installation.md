# 安装指南

本指南将帮助你完成项目的安装和配置。

## 系统要求

- Python 3.8+
- Node.js 16+
- Git 2.0+

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/username/repo.git
cd repo
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 即可查看效果。
