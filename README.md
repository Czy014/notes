# GitHub Pages 测试仓库

这是一个用于测试 GitHub Pages 功能的示例仓库。

## 目录结构

```
.
├── _config.yml       # Jekyll 配置文件
├── README.md         # 本文件
├── index.md          # 首页
├── about.md          # 关于页面
└── docs/             # 文档目录
    ├── index.md      # 文档中心首页
    ├── installation.md  # 安装指南
    ├── api.md        # API 参考
    └── examples.md   # 示例代码
```

## 如何使用

1. 将此仓库推送到 GitHub
2. 在仓库设置中开启 GitHub Pages
3. 访问你的站点

## 特性

- ✅ Markdown 内容渲染
- ✅ 代码高亮
- ✅ 数学公式支持（KaTeX）
- ✅ 表格和列表
- ✅ 响应式主题
